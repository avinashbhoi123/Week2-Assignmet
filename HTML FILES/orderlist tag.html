    Tags

    1. title

      Sets the title of the HTML document, which is displayed in the browser's title bar or tab. Provides a brief description of the page's content.

    2. link

      Defines external resources like stylesheets (CSS) and favicons. Allows for the inclusion of external files to enhance the appearance and functionality of the page.

    3. style

      Contains CSS code to define the presentation and styling of HTML elements. Can be placed in the head or directly within an HTML document.

    4. nav

      Defines a section of navigation links, such as menus or lists of links to other pages. Helps users navigate through the website.
